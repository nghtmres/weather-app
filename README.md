# Weather App

## About

- A simple command-line weather application built with Python. 
- Users can enter any city and receive current weather information, including temperature, humidity, wind speed, and location details.

## Features

- Search weather by city
- Display country
- Display latitude and longitude
- Display current temperature
- Display current relative humidity
- Display current wind speed
- Error handling

## Technologies

- Python
- Requests
- Open-Meteo API

## How to run

1. Clone repository
2. Navigate to the project folder
3. Run:

 ```bash
 python main.py
 ```
